**Чтобы начать:**

- Клонируй репозиторий к себе на компьютер
- Установи все зависимости: npm install
- Запусти локальный сервер: npm run dev

https://free-stylo-test-without-local-storage.vercel.app/

- TypeScript
- NextJs
- reduxjs/toolkit 
- Scss

- axios
- localStorage для хранения данных
